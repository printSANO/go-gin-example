# Go 언어로 만드는 Rest API Boilerplate

> 이 레포는 테커 GoodNight Hackathon 3차를 위해 만들어진 Boilerplate 레포 입니다

## 사용된 프레임워크와 기술들
- Go 1.23
- Go Gin
- Gorm
- PostgreSQL
- Docker

